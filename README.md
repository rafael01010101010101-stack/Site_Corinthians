# Portal Temático - Sport Club Corinthians Paulista 🦅⚽

Este é um projeto de desenvolvimento web completo (Full-Stack básico) focado na criação de uma interface funcional, altamente dinâmica e totalmente dedicada ao Corinthians. A aplicação combina uma interface feita à mão com lógica de programação no navegador e processamento de dados no servidor para entregar um portal interativo completo.

---

## 🎨 Destaques do Projeto
- **Design Temático e Identidade Visual:** Uso fiel das cores oficiais, tipografia e elementos visuais que remetem à forte identidade do clube.
- **Interatividade e Dinamismo:** Recursos em tempo real para melhorar a experiência do usuário durante a navegação pelas notícias, história e elenco.
- **Processamento no Servidor:** Integração com lógica de back-end para gerenciar o envio de formulários (como páginas de contato ou cadastro de torcedores) de forma segura.

## 🛠️ Stack Tecnológica
- **HTML5:** Estruturação semântica e organizada de todo o conteúdo e seções do site.
- **CSS3:** Estilização customizada, definição de layout e identidade visual feita totalmente do zero.
- **JavaScript:** Scripts de front-end para manipulação do DOM, criando animações, validações e comportamentos interativos.
- **PHP:** Linguagem de back-end utilizada para receber, validar e processar os dados enviados pelo usuário para o servidor.
- **Ambiente:** Desenvolvido utilizando o Visual Studio Code.

---

## 📈 Aprendizados Consolidados
Desenvolver este portal completo foi uma excelente oportunidade para evoluir competências Full-Stack:
1. **Lógica de Programação Ponta a Ponta:** Aplicação prática de variáveis, funções e condicionais tanto do lado do cliente (JavaScript) quanto do lado do servidor (PHP).
2. **Integração de Arquitetura Web:** Compreensão de como o front-end estruturado em HTML/CSS conversa com os scripts de processamento em PHP.
3. **Casamento entre Estética e Código:** Traduzir uma identidade visual forte e cheia de elementos em linhas de código organizadas, limpas e perfeitamente funcionais.
