# Portal Temático - Sport Club Corinthians Paulista 🦅⚽

Este é um projeto de desenvolvimento web focado na criação de uma interface funcional, visualmente atraente e totalmente dedicada ao Corinthians. O objetivo principal foi consolidar conhecimentos de estruturação de páginas, estilização customizada e semântica web.

---

## 🎨 Destaques do Projeto
- **Design Temático e Identidade Visual:** Uso fiel das cores oficiais, tipografia e elementos visuais que remetem à identidade do clube.
- **Estruturação Semântica:** Código limpo e organizado utilizando as melhores práticas para garantir uma boa leitura por navegadores.
- **Layout Customizado:** Organização manual de componentes visuais, menus de navegação e seções de conteúdo (como notícias, história e elenco).

## 🛠️ Tecnologias Utilizadas
- **HTML5:** Para a estruturação semântica de todo o conteúdo do site.
- **CSS:** Para a estilização, definição de layout, cores e responsividade básica.
- **Ambiente:** Desenvolvido utilizando o Visual Studio Code.

---

## 📈 Aprendizados Consolidados
Desenvolver este portal foi uma excelente oportunidade para praticar habilidades fundamentais de front-end:
1. **Manipulação de Box Model e Posicionamento:** Domínio sobre como alinhar elementos, imagens e textos de forma harmônica na tela.
2. **Organização de Arquivos de Mídia:** Gerenciamento correto de diretórios de imagens, folhas de estilo e caminhos relativos dentro do projeto.
3. **Casamento entre Estética e Código:** Traduzir uma paixão e uma identidade visual forte em linhas de código organizadas.
